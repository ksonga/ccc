This course is about writing computer programs using Java. In the real world, programs are becoming essential in everyday’s activities. To create programs or apps that are used in Hospitality, banking, health, education, public sector, etc. you will need to learn at least one programming language. This course will introduce you to the different programming concepts including the various programming languages that are commonly used. You will get a chance to focus on Java and learn the fundamental principles of programming like how to declare variables, use control structures, create objects and classes using Java. This will give a firm foundation for future courses that will require the use of Java and programming knowledge in general. It will also prepare you and put you on the right track to becoming the software engineer.
